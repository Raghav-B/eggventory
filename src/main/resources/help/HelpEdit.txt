Directly modifies the value of a property of an object.

edit:       edit stock <Stock Code> <Property> <New Value>
            edit stocktype <Stock Type> <New Stock Type Name>
            edit person <MatricNo> <Property> <New Value>

<Stock Code>        The model number of the existing stock
<Stock Type>        
<Property>          The property of the object to change
<New Value>


<New Value>         The value of the property mentioned will be replaced by this new value

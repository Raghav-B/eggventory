Directly modifies the value of a property of an object.

edit:       edit stock <StockCode> <Property> <NewValue>
            edit stocktype <StockType> <NewStockTypeName>
            edit person <MatricNo> <Property> <NewValue>

<StockCode>        The model number of the existing stock
<StockType>
<Property>          The property of the object to change
<NewValue>


<NewValue>         The value of the property mentioned will be replaced by this new value
